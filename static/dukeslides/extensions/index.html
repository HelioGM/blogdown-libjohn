The following are CSS style extensions to the xaringan (or xaringan -- dukeslides) presentations styles

  • Orange Splash page
    • This extension is intended to provide an alternative opening screen (the `.inverse` option) to the `dukeslides` default of a dark duke blue. This style should be used simultaneously with the duke-blue style
  • Captions for Figures leverages the figcaptions CSS style
  • Creative Commons licensing CSS style and fonts
